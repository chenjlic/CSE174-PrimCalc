# CSE174-PrimCalc
Primitive Calculator
Program that takes in a input file and calculates the result of the input and outputs to terminal.
Input file has to be formatted like so:
+ 1 2
- 10 3
+ -1 -1
* 2 5
Output would be:
Result of Line 1: 3
Result of Line 2: 7
Result of Line 3: -2
Result of Line 4: 10
